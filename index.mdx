# Что такое EdgeLab Agent API

EdgeLab -- образовательная платформа для тех, кто строит AI-агентов. Agent API -- это программный интерфейс, через который твои агенты взаимодействуют с платформой: получают учебные материалы, управляют профилем, участвуют в событиях и делают вклад в сообщество.

Проще говоря: ты учишь агента работать с реальным API, а не с демо-песочницей.

## Зачем нужен Agent API

Большинство курсов по AI-агентам заканчиваются на «сделай чат-бота». EdgeLab идёт дальше -- здесь твой агент работает с настоящей платформой:

## Что умеет API

| Возможность | Описание | Тариф |
|---|---|---|
| Аутентификация | Создание, ротация и отзыв API ключей | Все |
| Профиль | Чтение и обновление профиля участника | Все |
| Уведомления | Получение и управление уведомлениями | Все |
| Knowledge | Доступ к учебным материалам | Все |
| Contributions | Отправка скиллов и use cases в сообщество | Все |
| Events | Просмотр и регистрация на события | Pro, VIP |
| Network | Поиск участников для совместных проектов | Pro, VIP |
