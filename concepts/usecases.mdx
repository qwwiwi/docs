# Каталог Use Cases

EdgeLab собирает проверенные pipeline и скиллы для AI-агентов. Каждый use case прошёл проверку качества и протестирован на реальных задачах.

## Use Cases API

Use Cases API -- это отдельный MCP-сервер, который даёт AI-агентам доступ к внешним инструментам и данным через единую точку подключения. Ты подсоединяешь своего агента к MCP и получаешь готовые tools для работы с реальными сервисами.

<Info>
  Use Cases API находится в активной разработке. Новые инструменты добавляются регулярно.
</Info>

### Первые инструменты

| Инструмент | Что делает | Статус |
|---|---|---|
| Instagram Data | Вытягивание любых данных из Instagram: профили, посты, рилсы, статистика, аналитика аккаунтов | В разработке |
| Twitter / X Reader | Чтение статей, тредов, профилей и поиск по Twitter/X | В разработке |

### Как это будет работать

1. Подключаешь Use Cases MCP к своему агенту (отдельный endpoint)
2. Агент получает tools для работы с Instagram, Twitter и другими сервисами
3. Вызываешь нужный tool -- получаешь данные в структурированном виде

Подробная документация появится после запуска первых инструментов.

## Хочешь добавить свой use case?

EdgeLab собирает не абстрактные идеи, а реальные рабочие схемы. Если ты уже автоматизировал контент, трафик, аналитику, разработку, продажи, клиентскую работу или внутреннюю операционку -- оформи это как contribution.

Подойдут:
- skills для AI-агентов
- use cases с пошаговой логикой
- pipeline для реальных задач
- уроки, гайды и шаблоны автоматизации

Отправить contribution можно через MCP tool `submit_contribution` ([подробнее](/api-reference/contributions)) или через веб-интерфейс платформы. Лучшие материалы участвуют в Bounty-программе EdgeLab.

## Как использовать

1. Выбери use case из каталога
2. Установи необходимые скиллы через EdgeLab API
3. Настрой параметры под свою задачу
4. Запусти pipeline

Каждый use case содержит инструкцию по установке, настройке и примеры использования.
