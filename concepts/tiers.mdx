# Тарифы

EdgeLab предлагает три тарифа. Каждый открывает доступ к определённому набору функций платформы и API scopes.

## Сравнение тарифов

| Возможность | Edge | Pro | VIP |
|---|:---:|:---:|:---:|
| **Цена** | 19 900 руб / 6 мес | 29 900 руб / 6 мес | 199 900 руб / год |
| Профиль (чтение/запись) | + | + | + |
| Уведомления | + | + | + |
| Knowledge (учебные материалы) | + | + | + |
| Contributions | + | + | + |
| Обращения в поддержку | + | + | + |
| Network (участники сообщества) | -- | + | + |
| Events (события) | -- | + | + |
| Создание событий | -- | -- | + |
| Premium поддержка | -- | -- | + |

## Edge

**19 900 руб за 6 месяцев**

Базовый тариф для тех, кто начинает строить агентов. Включает всё необходимое для обучения:

- Доступ к учебным материалам (Knowledge)
- Управление профилем
- Отправка contributions в сообщество
- Уведомления
- Обращения в поддержку

### API scopes

```
profile:read, profile:write, support:write, knowledge:read, contributions:write
```

Тариф Edge -- хороший старт. Ты получаешь доступ ко всем учебным материалам и можешь вносить вклад в сообщество. Если позже понадобится нетворкинг или события -- можно перейти на Pro.

## Pro

**29 900 руб за 6 месяцев**

Для тех, кто хочет больше практики и взаимодействия с сообществом:

- Всё из Edge
- Просмотр списка участников (Network)
- Создание запросов на нетворкинг
- Просмотр и регистрация на события (воркшопы, лайвы, митапы)
- Мульти-агент сценарии

### API scopes

```
profile:read, profile:write, support:write, knowledge:read, contributions:write,
network:read, network:write, events:read
```

Pro подходит, когда ты уже разобрался с основами и хочешь строить более сложных агентов, работать в связке с другими участниками или участвовать в событиях.

## VIP

**199 900 руб за год**

Полный доступ ко всем функциям платформы:

- Всё из Pro
- Создание и управление событиями
- Административные функции
- Premium поддержка

### API scopes

```
profile:read, profile:write, support:write, knowledge:read, contributions:write,
network:read, network:write, events:read, events:write
```

VIP -- для тех, кто строит агентов на профессиональном уровне и хочет максимум возможностей.

## Смена тарифа

Смена тарифа доступна в настройках аккаунта на [platform.edgelab.su](https://platform.edgelab.su). При повышении тарифа новые scopes активируются для всех существующих API ключей автоматически.
