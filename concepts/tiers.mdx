# Тарифы

В EdgeLab доступ к MCP зависит от тарифа. Важно: это не REST API и не набор endpoint'ов. Платформа работает через MCP-сервер `https://mcp.edgelab.su/mcp`, а набор доступных tools определяется твоим уровнем доступа.

<Info>
Цены в этой документации не публикуются. Если нужен актуальный состав планов и статусы подписок, смотри `get_billing_plans`.
</Info>

## Free

Подходит, если ты только знакомишься с EdgeLab и хочешь посмотреть базу гайдов без авторизации.

- Scope: `knowledge:read`
- Доступный контент: только `guide`
- Rate limit: `30/min`
- Видимые публичные tools: `health`, `search_knowledge`, `get_knowledge`, `list_knowledge`, `list_categories`, `get_knowledge_children`, `list_recent_knowledge`, `get_sections`, `get_dashboard`, `get_faq`, `get_billing_plans`, `list_events`, `get_event`, `get_event_calendar`, `register`

Free работает даже без ключа. Если клиент умеет подключаться к MCP-серверу без заголовков авторизации, ты уже можешь читать публичную часть базы знаний.

## Edge

Полноценный рабочий тариф: обучение, профиль, прогресс, нетворк, contributions, поддержка, уведомления, рефералы и RSVP на события. Для предложения событий, use cases и API Use Cases нужен Pro.

- Scopes: `21`
- Контент: `guide`, `lesson`, `skill`, `workshop`, `live`
- Rate limit: `60/min`

## Pro

Всё из Edge плюс use cases, API Use Cases (доступ к API-ключам сервисов: Instagram, Twitter, Crypto и другие) и предложение событий.

Для более активного участия в экосистеме. К Edge добавляется право предлагать события и открывается категория `usecase`.

- Scopes: `22`
- Контент: `guide`, `lesson`, `skill`, `workshop`, `usecase`, `live`
- Rate limit: `120/min`
- Extras: priority support, future USECASES API

## VIP

VIP даёт те же tools что Pro, но с rate limit 300/min и ранним доступом. Сейчас тариф не продаётся.

- Scopes: `22` (как Pro)
- Контент: полный доступ + early access
- Rate limit: `300/min`
- Extras: office hours, 1-on-1, early access

<Warning>
VIP сейчас деактивирован: планы `vip-month` и `vip-year` помечены как `is_active=false`. В документации он оставлен как модель доступа, но оформить его сейчас нельзя.
</Warning>

## Что реально видит агент

| Уровень | Tools | Что получает агент |
|---|---|---|
| Free | 15 | Публичный поиск по гайдам, события, FAQ, регистрация |
| Edge | 66 | Всё для работы: профиль, прогресс, нетворк, contributions, поддержка, уведомления, рефералы, RSVP |
| Pro | 68 | Всё из Edge + предложение событий + use cases + API Use Cases |
| VIP | 68 | Всё из Pro + rate limit 300/min + early access |

## Как выбрать

<AccordionGroup>

<Accordion title="Мне нужно просто посмотреть, что внутри">
Бери **Free**. Подключаешь MCP без ключа и читаешь гайды.

<Accordion title="Мне нужен рабочий доступ для себя и агента">

</Accordion>

<Accordion title="Мне нужен максимум публичных материалов и возможность предлагать события">

</Accordion>

<Accordion title="Почему VIP описан, но недоступен">
Потому что модель доступа существует, но продажа этих планов сейчас выключена. Доки должны это явно отражать.
