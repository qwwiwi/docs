# Тарифы

В EdgeLab доступ к MCP зависит от тарифа. Важно: это не REST API и не набор endpoint'ов. Платформа работает через MCP-сервер `https://mcp.edgelab.su/mcp`, а набор доступных tools определяется твоим уровнем доступа.

<Info>
Цены в этой документации не публикуются. Если нужен актуальный состав планов и статусы подписок, смотри `get_billing_plans`.
</Info>

## Free

Подходит, если ты только знакомишься с EdgeLab и хочешь посмотреть базу гайдов без авторизации.

- Scope: `knowledge:read`
- Доступный контент: только `guide`
- Rate limit: `30/min`
- Видимые публичные tools: `search_knowledge`, `list_knowledge`, `get_sections`, `get_faq`, `health`, `get_billing_plans`, `register`, `email_login`

Free работает даже без ключа. Если клиент умеет подключаться к MCP-серверу без заголовков авторизации, ты уже можешь читать публичную часть базы знаний.

## Edge

Это основной рабочий уровень для человека и его агента: обучение, профиль, прогресс, события, поддержка, нетворк и contributions уже доступны.

- Scopes: `21`
- Контент: `guide`, `lesson`, `skill`, `workshop`, `live`
- Rate limit: `60/min`

## Pro

Для более активного участия в экосистеме. К Edge добавляется право предлагать события и открывается категория `usecase`.

- Scopes: `22`
- Контент: `guide`, `lesson`, `skill`, `workshop`, `usecase`, `live`
- Rate limit: `120/min`
- Extras: priority support, future USECASES API

## VIP

VIP даёт полный контент, высокий лимит запросов и ранний доступ, но сейчас этот тариф в продажу не выдаётся.

- Scopes: `20`
- Контент: полный доступ + early access
- Rate limit: `300/min`
- Extras: office hours, 1-on-1, early access

<Warning>
VIP сейчас деактивирован: планы `vip-month` и `vip-year` помечены как `is_active=false`. В документации он оставлен как модель доступа, но оформить его сейчас нельзя.
</Warning>

## Что реально видит агент

| Уровень | Что получает агент |
|---|---|
| Free | Публичный поиск по гайдам и FAQ без авторизации |
| Edge | Полноценную личную работу: профиль, прогресс, очки, события, support, network, contributions |
| Pro | Всё из Edge + `events:write`, категория `usecase`, priority support |
| VIP | Всё из Pro, кроме referral scopes, плюс early access и расширенные лимиты |
