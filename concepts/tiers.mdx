# Тарифы

EdgeLab предлагает три тарифа. Каждый открывает доступ к определённому набору функций платформы и API scopes.

## Сравнение тарифов

| Возможность | Edge | Pro | VIP |
|---|:---:|:---:|:---:|
| Профиль (чтение/запись) | + | + | + |
| Уведомления | + | + | + |
| Knowledge (учебные материалы) | + | + | + |
| Network (поиск участников) | + | + | + |
| Events (просмотр событий) | + | + | + |
| Support (тикеты) | -- | + | + |
| Contributions (скиллы, уроки) | -- | + | + |
| Network (connect-запросы) | -- | + | + |
| Предложить событие | -- | + | + |
| Выделенный VIP-чат с командой | -- | -- | + |
| Персональный созвон (1 в месяц) | -- | -- | + |
| Приоритетная обработка тикетов | -- | -- | + |

</Info>

## Edge

Базовый тариф для тех, кто начинает строить агентов. Включает всё необходимое для обучения:

- Доступ ко всем учебным материалам (Knowledge)
- Управление профилем
- Просмотр участников сообщества и событий
- Уведомления

### API scopes

```
profile:read, profile:write, knowledge:read, events:read, network:read
```

Edge -- хороший старт. Ты получаешь доступ ко всем учебным материалам, видишь участников и события. Когда понадобится нетворкинг, поддержка или contributions -- переходи на Pro.

## Pro

Для тех, кто хочет больше практики и взаимодействия с сообществом:

- Всё из Edge
- Создание connect-запросов (нетворкинг)
- Предложение событий (предложение проходит модерацию -- администратор решает)
- Отправка contributions (скиллы, уроки, use cases)
- Обращения в поддержку (тикеты через @EdgeLabSupportBot и API)
- Мульти-агент сценарии

### API scopes

```
profile:read, profile:write, knowledge:read, events:read, network:read,
support:write, contributions:write, events:write, network:write
```

Pro подходит, когда ты уже разобрался с основами и хочешь строить более сложных агентов, работать в связке с другими участниками и вносить вклад в экосистему.

## VIP

Полный доступ ко всем функциям платформы плюс персональное сопровождение:

- Всё из Pro
- **Выделенный VIP-чат с командой** -- отдельный чат, где присутствует команда EdgeLab
- **Персональный созвон** -- один созвон в месяц (консультация от команды)
- **Приоритетная обработка тикетов** -- тикеты VIP-участников обрабатываются в первую очередь

### API scopes

```
profile:read, profile:write, knowledge:read, events:read, network:read,
support:write, contributions:write, events:write, network:write, support:call-request
```

VIP -- для тех, кто строит агентов на профессиональном уровне и хочет персональное сопровождение.

## Поддержка по тарифам

| Возможность | Edge | Pro | VIP |
|---|:---:|:---:|:---:|
| FAQ (публичный) | + | + | + |
| Тикеты через @EdgeLabSupportBot и API | -- | + | + |
| Приоритетная обработка тикетов | -- | -- | + |
| Выделенный VIP-чат с командой | -- | -- | + |
| Персональный созвон (1 в месяц) | -- | -- | + |

**Pro и VIP** -- обращения в поддержку через тикеты. Без разницы в приоритете между Pro и VIP при обработке тикетов (кроме VIP-приоритета).

**VIP** -- дополнительно: отдельный чат с командой и один персональный созвон в месяц. Тикеты обрабатываются приоритетно.

## Смена тарифа

Смена тарифа доступна в настройках аккаунта на [platform.edgelab.su](https://platform.edgelab.su). При повышении тарифа новые scopes активируются для всех существующих API ключей автоматически.
