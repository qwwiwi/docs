# Roadmap

EdgeLab развивается итеративно. Вот что мы строим и что запланировано.

## Сейчас (Q1 2026)

<Steps>
  <Step title="Платформа и EdgeLab API v1">
    Учебные материалы, профили участников, уведомления, API ключи. Работает.
  </Step>
  <Step title="Contributions API">
    Отправка скиллов и use cases в сообщество. Проверка качества. Работает.
  </Step>
  <Step title="Документация">
    docs.edgelab.su -- полная документация API, гайды, концепции. Работает.
  </Step>
  <Step title="Support Bot">
    @EdgeLabSupportBot -- поддержка через Telegram. Работает.
  </Step>
</Steps>

## Следующий шаг (Q2 2026)

| Фича | Описание | Статус |
|---|---|---|
| EdgeLab API v2 | Расширенные endpoints: Network, Events, расширенный профиль | В разработке |
| Маркетплейс скиллов | Каталог с поиском, фильтрами и рейтингами | Планируется |
| Revenue Share | Автоматическое распределение вознаграждений contributors | Планируется |
| Лидерборд | Рейтинг участников по поинтам и contributions | Планируется |
| Нетворкинг v2 | Запросы на сотрудничество, совместные проекты | Планируется |

## Дальний горизонт

- **Маркетплейс агентов** -- готовые агенты под конкретные задачи
- **Multi-agent workflows** -- координация нескольких агентов в одном pipeline
- **Enterprise API** -- для команд и компаний
- **Mobile приложение** -- управление агентами с телефона
- **Интеграции** -- Slack, Notion, Telegram, Discord, GitHub

## Как влиять на roadmap

Участники Pro и VIP могут голосовать за приоритеты фичей. Contributors могут предлагать новые направления через Contributions API.
