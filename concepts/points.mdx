# Система поинтов

Поинты -- это внутренняя мера вклада участника в сообщество EdgeLab. Они показывают активность и помогают строить лидерборды, рекомендации и персональные отчёты.

## Как заработать поинты

| Действие | Поинты | Статус |
|---|---|---|
| Завершение урока | +10 | Работает |
| Отправка contribution | +25 | Работает |
| Создание тикета | +5 | Работает |
| Заполнение профиля | +15 | Работает |
| Отправка connect-запроса | +5 | Работает |
| Endorsement | +2 | Планируется |
| Одобрение contribution | +10 | Планируется |
| Featured contribution | +25 | Планируется |

## Пример

1. Заполнил профиль -> `+15`
2. Прошёл урок -> `+10`
3. Отправил contribution -> `+25`
4. Создал тикет -> `+5`
5. Отправил connect-запрос -> `+5`

Итого: `60` поинтов за базовую активность.

## Лидерборд

Лидерборд показывает топ участников по количеству поинтов:

- за неделю
- за месяц
- за всё время

Обычно в нём видны:

- ранг
- имя
- поинты
- contributions
- одобренные материалы

# Bounty программа

Bounty программа -- это отдельный слой поощрения за сильный вклад в EdgeLab.

<Info>
Размер фонда и конкретные условия в этой документации не публикуются. Здесь фиксируется только логика программы, без сумм и цен.
</Info>

## За что может начисляться bounty

- создание и обновление skills
- новые usecases и pipeline
- уроки и воркшопы
- системное улучшение базы знаний
- полезные инициативы внутри сообщества

## Как это работает

1. Команда смотрит на практическую ценность вклада
2. Оценивает качество упаковки и повторяемость результата
3. Сравнивает интерес и пользу для сообщества
4. Принимает решение о вознаграждении

## Что повышает шансы

- contribution решает реальную задачу
- его можно воспроизвести
- он полезен не одному человеку, а группе участников
- описание ясное и без воды

<Warning>
Bounty -- это не зарплата и не обещание фиксированной выплаты. Это отдельное вознаграждение за сильный вклад в экосистему.
</Warning>
