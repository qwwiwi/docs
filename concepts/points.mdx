# Система поинтов

Поинты -- это мера вклада участника в сообщество EdgeLab. Ты зарабатываешь поинты, создавая полезные contributions: скиллы, use cases, предложения событий. Поинты отображаются в профиле и в публичном лидерборде.

## Как заработать поинты

| Действие | Поинты | Описание |
|---|---|---|
| Отправка contribution | +1 | За каждый отправленный contribution, независимо от результата ревью |
| Endorsement | +2 | За каждый голос от другого участника за твой contribution |
| Одобрение (approve) | +10 | Contribution прошёл ревью и опубликован |
| Featured | +25 | Contribution отмечен как лучший |

## Пример подсчёта

Ты создал скилл для работы с погодным API:

1. Отправил contribution → **+1** поинт
2. 7 участников проголосовали за (endorsed) → **+14** поинтов
3. Администратор одобрил → **+10** поинтов
4. Через неделю скилл стал featured → **+25** поинтов

Итого: **50 поинтов** за один contribution.

## Лидерборд (запуск Q2 2026)

Лидерборд -- публичный рейтинг участников по количеству поинтов. Показывает top contributors за:

- Последнюю неделю
- Последний месяц
- Всё время

### Что отображается

| Поле | Описание |
|---|---|
| Ранг | Позиция в рейтинге |
| Имя | username и display_name участника |
| Поинты | Общее количество поинтов за выбранный период |
| Contributions | Количество отправленных contributions |
| Одобрено | Количество одобренных contributions |
| Featured | Количество featured contributions |

# Bounty программа (запуск Q2 2026)

Bounty программа -- вознаграждение участников сообщества, которые вносят вклад в развитие EdgeLab.

<Info>
Bounty программа доступна участникам на тарифах **Pro** и **VIP**.
</Info>

## За что вознаграждение

- Создание и обновление скиллов для AI-агентов
- Новые use cases и готовые pipeline
- Запись уроков и образовательных материалов
- Инициативы внутри сообщества (организация событий, менторство, помощь другим участникам)
- Улучшение существующих материалов

## Как устроен фонд

Для первых участников EdgeLab запускает стартовую Bounty-программу.

На ближайшие **2 месяца** выделен фонд **5 000 USDT** для первых сильных контрибьюторов, которые приносят в сообщество реальные полезные contributions:

- use cases
- skills для AI-агентов
- готовые pipeline
- уроки и практические материалы
- улучшения существующей базы знаний

Этот фонд будет распределён между участниками, которые реально помогают развивать экосистему EdgeLab своими рабочими решениями.

В дальнейшем Bounty-программа будет поддерживаться из бюджета сообщества и revenue EdgeLab.

## Как работает распределение

1. На старте фиксируется общий фонд: **5 000 USDT**
2. Период первой программы: **2 месяца**
3. Внутри этого периода команда оценивает вклад участников: качество contributions, практическую пользу, применение внутри сообщества и общий вклад в экосистему
4. По итогам периода фонд распределяется между сильнейшими контрибьюторами

### Что влияет на решение

- практическая полезность contribution
- качество упаковки и понятность материала
- возможность повторить use case в реальной работе
- отзывы, интерес и применение со стороны сообщества
- общий вклад участника в развитие EdgeLab

### Пример

Если участник приносит сильный use case, полезный skill или рабочий pipeline, который помогает другим людям и агентам быстрее запускать реальные AI-сценарии, такой вклад может участвовать в распределении фонда.

Чем сильнее contribution и чем больше практической пользы он приносит сообществу, тем выше вероятность получить bounty.

## Выплаты

- **Валюта:** USDT (криптовалюта)
- **Периодичность:** ежемесячно
- **Минимальный порог:** устанавливается администрацией

<Warning>
Bounty -- это вознаграждение за вклад в сообщество, а не зарплата или трудовой договор. Это донат от проекта для самых активных участников.
</Warning>

## Стратегия набора поинтов

Несколько советов:

- **Качество важнее количества.** Один одобренный и endorsed contribution даёт больше поинтов, чем пять отклонённых.
- **Решай реальные задачи.** Скиллы и use cases, которые решают конкретные проблемы, получают больше endorsements.
- **Используй feedback.** Если администратор запросил ревизию -- доработай и отправь снова. Это путь к approve.
- **Голосуй за других.** Endorsement -- это не только поинты для автора, но и участие в жизни сообщества.

<Note>
Стартовая Bounty-программа уже описана как механизм привлечения сильных контрибьюторов. Дальнейшая система будет расширяться вместе с ростом сообщества и базы contributions.
</Note>
