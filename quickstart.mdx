# Быстрый старт

В EdgeLab нет REST API для агентских интеграций. Быстрый старт выглядит так: подключаешь MCP-сервер, при желании добавляешь ключ и вызываешь tools.

<Steps>

<Step title="Подключись к MCP-серверу">

Адрес один:

```text
https://mcp.edgelab.su/mcp
```

Если тебе нужны только публичные гайды и FAQ, этого уже достаточно.

</Step>

<Step title="Выбери режим: public или personal">

| Режим | Что нужно | Что получаешь |
|---|---|---|
| Public | Ничего, даже ключ не нужен | Публичные tools и гайды |
| Personal | Ключ от `@InvestorEdgeLabbot` или OAuth через Telegram | Профиль, прогресс, events, network, support, contributions и другое |

</Step>

<Step title="Добавь конфиг в свой клиент">

<CodeGroup>
```bash Claude Code (CLI)
claude mcp add edgelab-mcp --transport http https://mcp.edgelab.su/mcp
```

```json Claude Code (конфиг)
{
  "mcpServers": {
    "edgelab": {
      "type": "streamable-http",
      "url": "https://mcp.edgelab.su/mcp"
    }
  }
}
```

```json Claude Desktop
{
  "mcpServers": {
    "edgelab": {
      "url": "https://mcp.edgelab.su/mcp",
      "headers": {
        "Authorization": "Bearer el_live_ваш_ключ"
      }
    }
  }
}
```

```json Cursor / Windsurf
{
  "mcpServers": {
    "edgelab": {
      "url": "https://mcp.edgelab.su/mcp",
      "headers": {
        "x-api-key": "el_live_ваш_ключ"
      }
    }
  }
}
```
</CodeGroup>

<Tip>
**Claude Code** не требует API-ключ — при первом запросе автоматически запустится OAuth через Telegram. Просто подтверди в боте и всё заработает.
</Tip>

</Step>

<Step title="Авторизуйся">

**Для Claude Code (OAuth):**
1. При первом вызове tool откроется браузер
2. На странице — кнопка **«Открыть Telegram»**
3. Перейди в бота, нажми START → готово

**Для остальных клиентов (API-ключ):**
1. Открой [@InvestorEdgeLabbot](https://t.me/InvestorEdgeLabbot)
2. Нажми START → бот выдаст ключ `el_live_...`
3. Добавь ключ в конфиг клиента

</Step>

<Step title="Сделай первый вызов">

Самый безопасный первый вызов без ключа:

```json
{
  "jsonrpc": "2.0",
  "id": "1",
  "method": "tools/call",
  "params": {
    "name": "list_knowledge",
    "arguments": {
      "category": "guide",
      "page": 1,
      "per_page": 5
    }
  }
}
```

Если всё подключено правильно, в ответе придёт список гайдов.

</Step>

</Steps>
