# Быстрый старт

В EdgeLab нет REST API для агентских интеграций. Быстрый старт выглядит так: подключаешь MCP-сервер, при желании добавляешь ключ и вызываешь tools.

<Steps>

<Step title="Подключись к MCP-серверу">

Адрес один:

```text
https://mcp.edgelab.su/mcp
```

Если тебе нужны только публичные гайды и FAQ, этого уже достаточно.

</Step>

<Step title="Выбери режим: public или personal">

| Режим | Что нужно | Что получаешь |
|---|---|---|
| Public | Ничего, даже ключ не нужен | Публичные tools и гайды |
| Personal | Ключ от `@InvestorEdgeLabbot` | Профиль, прогресс, events, network, support, contributions и другое |

Формат ключа:

```text
el_live_1a8c25f8b7165604
```

</Step>

<Step title="Добавь конфиг в свой клиент">

<CodeGroup>
```json Claude Desktop
{
  "mcpServers": {
    "edgelab": {
      "url": "https://mcp.edgelab.su/mcp"
    }
  }
}
```

```json Cursor
{
  "mcpServers": {
    "edgelab": {
      "url": "https://mcp.edgelab.su/mcp",
      "headers": {
        "Authorization": "Bearer el_live_1a8c25f8b7165604"
      }
    }
  }
}
```

```json Windsurf
{
  "mcpServers": {
    "edgelab": {
      "url": "https://mcp.edgelab.su/mcp",
      "headers": {
        "x-api-key": "el_live_1a8c25f8b7165604"
      }
    }
  }
}
```
</CodeGroup>

<Note>
Если клиент не умеет кастомные заголовки прямо в конфиге, используй public-режим или локальный прокси/обёртку, которая добавляет `Authorization`/`x-api-key`.
</Note>

</Step>

<Step title="Сделай первый вызов">

Самый безопасный первый вызов без ключа:

```json
{
  "jsonrpc": "2.0",
  "id": "1",
  "method": "tools/call",
  "params": {
    "name": "list_knowledge",
    "arguments": {
      "category": "guide",
      "page": 1,
      "per_page": 5
    }
  }
}
```

Если всё подключено правильно, в ответе придёт список гайдов.

</Step>

<Step title="Попробуй личный tool с ключом">

После регистрации у бота можно вызывать личные tools, например профиль:

```json
{
  "jsonrpc": "2.0",
  "id": "2",
  "method": "tools/call",
  "params": {
    "name": "get_user_profile",
    "arguments": {}
  }
}
```

Или посмотреть свой баланс поинтов:

```json
{
  "jsonrpc": "2.0",
  "id": "3",
  "method": "tools/call",
  "params": {
    "name": "get_points_balance",
    "arguments": {}
  }
}
```

</Step>

</Steps>
