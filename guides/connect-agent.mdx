# Как подключить агента к EdgeLab API

Этот гайд показывает, как сделать так, чтобы твой AI-агент мог самостоятельно работать с EdgeLab API: читать материалы, управлять профилем и делать contributions.

## Что понадобится

- Аккаунт на [EdgeLab](https://platform.edgelab.su)
- API ключ (см. [Быстрый старт](/quickstart))
- AI-агент, который умеет делать HTTP-запросы (через tool/function calling или напрямую)

## Архитектура подключения

Есть два основных подхода:

<CardGroup cols={2}>
  <Card title="Прямое подключение" icon="arrow-right">

## Прямое подключение

<Steps>

<Step title="Создай tool-описание для агента">

Твоему агенту нужно знать, какие API-запросы он может делать. Вот пример tool-описания для вызова `GET /profile/me`:

```json
{
  "name": "edgelab_get_profile",
  "description": "Получить профиль текущего пользователя на EdgeLab",
  "parameters": {
    "type": "object",
    "properties": {},
    "required": []
  }
}
```

</Step>

<Step title="Реализуй обработчик tool-вызовов">

Когда агент решает вызвать tool, твой код должен сделать соответствующий HTTP-запрос:

```python
import requests
import os

EDGELAB_API_KEY = os.environ.get("EDGELAB_API_KEY")
BASE_URL = "https://platform.edgelab.su/api/v1"

def handle_tool_call(tool_name, arguments):
    headers = {
        "Authorization": f"Bearer {EDGELAB_API_KEY}",
        "Content-Type": "application/json"
    }

    if tool_name == "edgelab_get_profile":
        response = requests.get(
            f"{BASE_URL}/profile/me",
            headers=headers
        )
        return response.json()

    elif tool_name == "edgelab_update_profile":
        response = requests.patch(
            f"{BASE_URL}/profile/me",
            headers=headers,
            json=arguments
        )
        return response.json()

    elif tool_name == "edgelab_get_notifications":
        response = requests.get(
            f"{BASE_URL}/notifications",
            headers=headers,
            params={"unread_only": arguments.get("unread_only", False)}
        )
        return response.json()
```

</Step>

<Step title="Храни ключ безопасно">

Никогда не вставляй API ключ прямо в код. Используй переменные окружения:

```bash
export EDGELAB_API_KEY="edgelab_live_a1b2c3d4e5f6a1b2c3d4e5f6a1b2c3d4"
```

Или файл `.secrets/edgelab.env`:

```
EDGELAB_API_KEY=edgelab_live_a1b2c3d4e5f6a1b2c3d4e5f6a1b2c3d4
```

Подробнее -- в гайде [Безопасность API ключей](/guides/api-key-security).

</Step>

<Step title="Добавь обработку ошибок">

Агент должен уметь обрабатывать ошибки API:

```python
def safe_api_call(method, url, **kwargs):
    """Обёртка для API-вызовов с обработкой ошибок."""
    headers = {
        "Authorization": f"Bearer {EDGELAB_API_KEY}",
        "Content-Type": "application/json"
    }

    try:
        response = requests.request(
            method, url, headers=headers, **kwargs
        )

        if response.status_code == 429:
            retry_after = int(response.headers.get("Retry-After", 60))
            return {
                "error": f"Rate limit. Подожди {retry_after} секунд.",
                "retry_after": retry_after
            }

        if response.status_code == 401:
            return {"error": "Невалидный API ключ. Проверь настройки."}

        if response.status_code == 403:
            return {"error": "Нет доступа. Возможно, нужен другой тариф."}

        return response.json()

    except requests.exceptions.RequestException as e:
        return {"error": f"Ошибка сети: {str(e)}"}
```

</Step>

<Step title="Протестируй интеграцию">

Запусти агента и проверь, что он может:

1. Получить свой профиль
2. Обновить описание профиля
3. Проверить уведомления
4. Корректно обработать ошибку (например, запрос с невалидным ключом)

</Step>

</Steps>

## Пример: минимальный агент

Вот полный пример простого агента, который умеет работать с EdgeLab API:

```python
import requests
import os

class EdgeLabAgent:
    def __init__(self):
        self.api_key = os.environ["EDGELAB_API_KEY"]
        self.base_url = "https://platform.edgelab.su/api/v1"

    def _request(self, method, path, **kwargs):
        """Базовый HTTP-запрос к API."""
        response = requests.request(
            method,
            f"{self.base_url}{path}",
            headers={
                "Authorization": f"Bearer {self.api_key}",
                "Content-Type": "application/json"
            },
            **kwargs
        )
        return response.json()

    def get_profile(self):
        """Получить свой профиль."""
        return self._request("GET", "/profile/me")

    def update_bio(self, bio):
        """Обновить описание профиля."""
        return self._request("PATCH", "/profile/me", json={"bio": bio})

    def get_unread_notifications(self):
        """Получить непрочитанные уведомления."""
        return self._request("GET", "/notifications",
                           params={"unread_only": True})

    def mark_notification_read(self, notification_id):
        """Отметить уведомление как прочитанное."""
        return self._request("PATCH",
                           f"/notifications/{notification_id}/read")


# Использование
agent = EdgeLabAgent()
profile = agent.get_profile()
print(f"Привет, {profile['data']['display_name']}!")
print(f"Тариф: {profile['data']['tier']}")
print(f"Поинты: {profile['data']['points']}")
```
