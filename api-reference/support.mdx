# Support

Поддержка в EdgeLab делится на две части: публичный FAQ и личные support tools.

## Public

| Tool | Доступ | Что делает |
|---|---|---|
| `get_faq` | Public | Возвращает FAQ |

FAQ можно читать даже без ключа. Это хороший первый шаг, прежде чем агент создаёт тикет.

## Personal support tools

| Tool | Доступ | Что делает |
|---|---|---|
| `create_ticket` | Pro/VIP | Создаёт тикет |
| `list_tickets` | Pro/VIP | Возвращает список тикетов |
| `get_ticket` | Pro/VIP | Возвращает один тикет и его контекст |
| `send_ticket_message` | Pro/VIP | Отправляет сообщение в тикет |

## `get_faq`

Подходит для автоматического self-help flow:

1. агент сначала вызывает `get_faq`
2. если подходящего ответа нет, идёт в support tools

## `create_ticket`

Создаёт новый тикет.

Типичный минимум:

- тема
- сообщение
- категория или контекст проблемы

## `list_tickets`

Нужен для фонового мониторинга открытых запросов и ожидания ответа.

## `get_ticket`

Возвращает детали конкретного тикета и историю обсуждения.

## `send_ticket_message`

Позволяет продолжить уже открытый тикет, не создавая новый.

### Практический flow

```text
get_faq -> create_ticket -> list_tickets -> get_ticket -> send_ticket_message
```
