# Knowledge endpoints

<Note>
Эти endpoints находятся в разработке и будут доступны в ближайшем обновлении. Ниже описана планируемая функциональность.
</Note>

Endpoints для доступа к учебным материалам EdgeLab: уроки, гайды, use cases. Материалы доступны в форматах markdown и HTML.

---

## GET /knowledge/items

<Info>
Coming soon -- этот endpoint появится в ближайшем обновлении API.
</Info>

Получить список доступных учебных материалов. Возвращает заголовки, описания и метаданные без полного содержимого.

**Scope:** `knowledge:read` (все тарифы)

**Полный URL:** `GET https://platform.edgelab.su/api/v1/knowledge/items`

### Планируемые query-параметры

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| `page` | integer | 1 | Номер страницы |
| `per_page` | integer | 20 | Количество на странице (макс. 100) |
| `category` | string | -- | Фильтр по категории: `lesson`, `guide`, `usecase` |
| `tier` | string | -- | Фильтр по минимальному тарифу: `edge`, `pro`, `vip` |

### Планируемая структура ответа

Каждый элемент списка будет содержать:

| Поле | Тип | Описание |
|---|---|---|
| `id` | string | Уникальный идентификатор материала |
| `title` | string | Заголовок |
| `description` | string | Краткое описание |
| `category` | string | Категория: `lesson`, `guide`, `usecase` |
| `min_tier` | string | Минимальный тариф для доступа |
| `author` | string | Автор материала |
| `created_at` | string | Дата публикации (ISO 8601) |
| `updated_at` | string | Дата последнего обновления (ISO 8601) |

---

## GET /knowledge/items/{id}

<Info>
Coming soon -- этот endpoint появится в ближайшем обновлении API.
</Info>

Получить полное содержимое учебного материала по идентификатору. Содержимое возвращается в формате markdown.

**Scope:** `knowledge:read` (все тарифы)

**Полный URL:** `GET https://platform.edgelab.su/api/v1/knowledge/items/{id}`

### Планируемые параметры пути

| Параметр | Тип | Описание |
|---|---|---|
| `id` | string | Идентификатор материала |

### Планируемые query-параметры

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| `format` | string | `markdown` | Формат содержимого: `markdown` или `html` |

### Планируемая структура ответа

| Поле | Тип | Описание |
|---|---|---|
| `id` | string | Идентификатор материала |
| `title` | string | Заголовок |
| `description` | string | Краткое описание |
| `content` | string | Полное содержимое в выбранном формате |
| `category` | string | Категория |
| `min_tier` | string | Минимальный тариф |
| `author` | string | Автор |
| `created_at` | string | Дата публикации |
| `updated_at` | string | Дата обновления |

### Планируемые ошибки

| HTTP код | Когда |
|---|---|
| 403 Forbidden | Тариф пользователя ниже `min_tier` материала |
| 404 Not Found | Материал с указанным id не найден |
