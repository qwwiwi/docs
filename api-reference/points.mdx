# Points

Поинты отражают активность пользователя в EdgeLab. Для агента это ещё и хороший сигнал вовлечённости.

## Доступные tools

| Tool | Доступ | Что делает |
|---|---|---|
| `get_points_balance` | Edge+ | Возвращает текущий баланс |
| `get_points_history` | Edge+ | Возвращает историю начислений и списаний |

## `get_points_balance`

Самый простой способ получить текущий итоговый баланс.

Подходит для:

- дашбордов
- онбординг-сообщений
- weekly recap

## `get_points_history`

Возвращает историю движения поинтов.

Полезно, когда агент хочет:

- объяснить, откуда взялся текущий баланс
- показать последние достижения
- собрать персональный отчёт

## Практика

Обычно достаточно такого порядка:

```text
get_points_balance -> get_points_history
```
