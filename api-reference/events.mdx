# Events endpoints

<Note>
Эти endpoints находятся в разработке и будут доступны в ближайшем обновлении. Ниже описана планируемая функциональность.
</Note>

Endpoints для работы с событиями платформы: воркшопы, митапы, лайвы. Просмотр списка и деталей событий, регистрация и отмена участия.

---

## GET /events

<Info>
Coming soon -- этот endpoint появится в ближайшем обновлении API.
</Info>

Получить список событий платформы.

**Scope:** `events:read` (Pro, VIP)

**Полный URL:** `GET https://platform.edgelab.su/api/v1/events`

### Планируемые query-параметры

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| `page` | integer | 1 | Номер страницы |
| `per_page` | integer | 20 | Количество на странице (макс. 100) |
| `status` | string | -- | Фильтр по статусу: `upcoming`, `live`, `past` |

### Планируемая структура ответа

Каждый элемент списка будет содержать:

| Поле | Тип | Описание |
|---|---|---|
| `id` | string | Уникальный идентификатор события |
| `title` | string | Название события |
| `description` | string | Описание |
| `status` | string | Статус: `upcoming`, `live`, `past` |
| `starts_at` | string | Дата и время начала (ISO 8601) |
| `ends_at` | string | Дата и время окончания (ISO 8601) |
| `max_participants` | integer | Максимальное число участников (null если без ограничений) |
| `registered_count` | integer | Количество зарегистрированных |
| `is_registered` | boolean | Зарегистрирован ли текущий пользователь |

### Планируемые ошибки

| HTTP код | Когда |
|---|---|
| 403 Forbidden | Тариф Edge -- нет scope `events:read` |

## GET /events/{id}

<Info>
Coming soon -- этот endpoint появится в ближайшем обновлении API.
</Info>

Получить детали конкретного события.

**Scope:** `events:read` (Pro, VIP)

**Полный URL:** `GET https://platform.edgelab.su/api/v1/events/{id}`

### Планируемые параметры пути

| Параметр | Тип | Описание |
|---|---|---|
| `id` | string | Идентификатор события |

### Планируемая структура ответа

Все поля из списка событий, плюс:

| Поле | Тип | Описание |
|---|---|---|
| `full_description` | string | Полное описание события (markdown) |
| `speakers` | array | Список спикеров |
| `location` | string | Место проведения или ссылка на трансляцию |
| `participants` | array | Список зарегистрированных участников (только имена) |

## POST /events/{id}/register

<Info>
Coming soon -- этот endpoint появится в ближайшем обновлении API.
</Info>

Зарегистрироваться на событие.

**Scope:** `events:read` (Pro, VIP)

**Полный URL:** `POST https://platform.edgelab.su/api/v1/events/{id}/register`

### Планируемые параметры пути

| Параметр | Тип | Описание |
|---|---|---|
| `id` | string | Идентификатор события |

### Планируемое поведение

- Если регистрация успешна -- возвращается подтверждение с деталями события
- Если пользователь уже зарегистрирован -- `409 Conflict`
- Если мест нет -- `409 Conflict` с сообщением о заполненности
- Если событие уже прошло -- `422 Validation Error`

## DELETE /events/{id}/register

<Info>
Coming soon -- этот endpoint появится в ближайшем обновлении API.
</Info>

Отменить регистрацию на событие.

**Scope:** `events:read` (Pro, VIP)

**Полный URL:** `DELETE https://platform.edgelab.su/api/v1/events/{id}/register`

### Планируемые параметры пути

| Параметр | Тип | Описание |
|---|---|---|
| `id` | string | Идентификатор события |

- Если отмена успешна -- подтверждение
- Если пользователь не был зарегистрирован -- `404 Not Found`
- Если событие уже началось или прошло -- `422 Validation Error`
